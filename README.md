# Trabalho Final AED – Análise Exploratória de Dados

Projeto acadêmico desenvolvido como trabalho final da disciplina de Análise Exploratória de Dados (AED).

O projeto utiliza o formato `.qmd` do Quarto para criação de um relatório reproduzível, contendo análise estatística, visualização de dados e interpretação dos resultados.

---

## 🎯 Objetivo
Realizar uma análise exploratória de dados a partir de um conjunto de dados fornecido, aplicando conceitos estatísticos e técnicas de visualização para extração de insights.

---

## 🛠️ Tecnologias Utilizadas
- Quarto (.qmd)
- Linguagem R
- tidyverse
- ggplot2

---

## 📊 Conteúdo do Projeto
- Limpeza e preparação dos dados
- Análise estatística descritiva
- Criação de gráficos (histogramas, distribuições, comparações)
- Interpretação dos resultados obtidos

---

## 📁 Estrutura dos Arquivos
- `Trabalho Final AED.qmd` → Código-fonte do relatório
- `Trabalho Final AED.html` → Relatório renderizado (HTML)

---
